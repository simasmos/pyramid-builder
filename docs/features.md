# Features & Backlog - Pyramid Builder

## MVP Features (Phase 1)
- [ ] 20x20 square grid game board
- [ ] Worker unit placement and movement (4-directional)
- [ ] Action point system (2 AP per worker per turn)
- [ ] Turn-based game loop
- [ ] Stone quarrying from deposits
- [ ] Stone block placement mechanics
- [ ] Basic UI (AP counter, selected worker info)

## Core Features (Phase 2)
- [ ] Terrain system (desert, stone deposit, water/Nile)
- [ ] 3x3 pyramid structure in center
- [ ] Stone inventory management per worker
- [ ] River generation and impassable water tiles
- [ ] Win condition detection (pyramid completion)
- [ ] Main menu and game restart
- [ ] Visual feedback for valid actions

## Enhanced Features (Phase 3)
- [ ] Random map generation with river patterns
- [ ] Multiple pyramid designs/challenges
- [ ] Efficiency scoring system (turns taken, stones used)
- [ ] Undo/redo action system
- [ ] Save/load game state
- [ ] Tutorial and hint system
- [ ] Sound effects for actions

## Polish Features (Phase 4)
- [ ] Smooth movement animations
- [ ] Egyptian-themed visual assets
- [ ] Ambient desert/river sounds
- [ ] Achievement system (speed runs, efficiency)
- [ ] Statistics tracking
- [ ] Mobile touch controls optimization
- [ ] Multiple difficulty levels (different map layouts)
